# FrameLab Desktop

FrameLab Desktop est une application de bureau développée dans le cadre de l'épreuve E6 du BTS SIO 2026 - Option SLAM.

Le projet a été conçu pour l'entreprise fictive FrameLab, spécialisée dans la photographie haut de gamme et l'organisation de challenges photo communautaires.

L'application permet aux utilisateurs de :

* télécharger automatiquement l'image du challenge en cours ;
* retoucher l'image avec les outils intégrés ;
* sauvegarder leurs projets localement ;
* envoyer leur participation directement via l'API REST de FrameLab Web.

FrameLab Desktop fonctionne en complément de FrameLab Web, la plateforme communautaire permettant de consulter les participations, voter, commenter et suivre les classements des contributeurs.
